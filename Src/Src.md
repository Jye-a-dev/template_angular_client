# Src

- ``Index.html``: HTML template for this area.
- ``Main.ts``: TypeScript implementation for this area.
- ``Main_Server.ts``: TypeScript implementation for this area.
- ``Server.ts``: TypeScript implementation for this area.
- ``Styles.css``: stylesheet for this area.

Use this folder for:

- Keep this concern organized and separate from unrelated code.
- Use this folder for implementation notes, structure notes, and supporting files for this area.

Nên dùng thư mục này cho:
- Chứa các file liên quan đến khu vực này tại đây.
