# Core

## English
Purpose: This folder defines the Core area inside Core.

Current contents:
- ``Guards/``: subfolder for guards related code and docs.
- ``Interceptors/``: subfolder for interceptors related code and docs.
- ``Models/``: subfolder for models related code and docs.
- ``Providers/``: subfolder for providers related code and docs.
- ``Services/``: subfolder for services related code and docs.
- ``Utils/``: subfolder for utils related code and docs.

Use this folder for:
- Store files related to this area here.
- Keep this concern organized and separate from unrelated code.
- Use this folder for implementation notes, structure notes, and supporting files for this area.

## Tiếng Việt
Mục đích: Thư mục này định nghĩa khu vực Core bên trong Core.

Nội dung hiện tại:
- ``Guards/``: thư mục con cho mã nguồn và tài liệu liên quan đến guards.
- ``Interceptors/``: thư mục con cho mã nguồn và tài liệu liên quan đến interceptors.
- ``Models/``: thư mục con cho mã nguồn và tài liệu liên quan đến models.
- ``Providers/``: thư mục con cho mã nguồn và tài liệu liên quan đến providers.
- ``Services/``: thư mục con cho mã nguồn và tài liệu liên quan đến services.
- ``Utils/``: thư mục con cho mã nguồn và tài liệu liên quan đến utils.

Nên dùng thư mục này cho:
- Chứa các file liên quan đến khu vực này tại đây.
- Giữ concern này được tổ chức rõ ràng và tách khỏi phần code không liên quan.
- Dùng thư mục này cho ghi chú triển khai, ghi chú cấu trúc và các file hỗ trợ của khu vực này.
