# App

## English
Purpose: This folder defines the App area inside App.

Current contents:
- ``(Dashboard)/``: subfolder for dashboard related code and docs.
- ``(Public)/``: subfolder for public related code and docs.
- ``Core/``: subfolder for core related code and docs.
- ``Features/``: subfolder for features related code and docs.
- ``Layouts/``: subfolder for layouts related code and docs.
- ``Shared/``: subfolder for shared related code and docs.
- ``App.html``: HTML template for this area.
- ``App.ts``: TypeScript implementation for this area.
- ``App_Config.ts``: TypeScript implementation for this area.
- ``App_Config_Server.ts``: TypeScript implementation for this area.
- ``App_Routes.ts``: TypeScript implementation for this area.
- ``App_Routes_Server.ts``: TypeScript implementation for this area.
- ``App_Spec.ts``: TypeScript implementation for this area.

Use this folder for:
- Store files related to this area here.
- Keep this concern organized and separate from unrelated code.
- Use this folder for implementation notes, structure notes, and supporting files for this area.

Mục đích: Thư mục này định nghĩa khu vực App bên trong App.

Nội dung hiện tại:
- ``(Dashboard)/``: thư mục con cho mã nguồn và tài liệu liên quan đến dashboard.
- ``(Public)/``: thư mục con cho mã nguồn và tài liệu liên quan đến public.

- ``Features/``: thư mục con cho mã nguồn và tài liệu liên quan đến features.
- ``Layouts/``: thư mục con cho mã nguồn và tài liệu liên quan đến layouts.
- ``Shared/``: thư mục con cho mã nguồn và tài liệu liên quan đến shared.
- ``App.html``: file template HTML cho khu vực này.
- ``App.ts``: file triển khai TypeScript cho khu vực này.
- ``App_Config.ts``: file triển khai TypeScript cho khu vực này.
- ``App_Config_Server.ts``: file triển khai TypeScript cho khu vực này.
- ``App_Routes.ts``: file triển khai TypeScript cho khu vực này.

Nên dùng thư mục này cho:
- Chứa các file liên quan đến khu vực này tại đây.
