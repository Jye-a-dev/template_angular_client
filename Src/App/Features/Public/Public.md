# Public

## English
Purpose: This folder defines the Public area inside Public.

Current contents:
- ``Auth_Page/``: subfolder for auth page related code and docs.
- ``Home_Page/``: subfolder for home page related code and docs.

Use this folder for:
- Store files related to this area here.
- Keep this concern organized and separate from unrelated code.
- Use this folder for implementation notes, structure notes, and supporting files for this area.

## Tiếng Việt
Mục đích: Thư mục này định nghĩa khu vực Public bên trong Public.

Nội dung hiện tại:
- ``Auth_Page/``: thư mục con cho mã nguồn và tài liệu liên quan đến auth page.
- ``Home_Page/``: thư mục con cho mã nguồn và tài liệu liên quan đến home page.

Nên dùng thư mục này cho:
- Chứa các file liên quan đến khu vực này tại đây.
- Giữ concern này được tổ chức rõ ràng và tách khỏi phần code không liên quan.
- Dùng thư mục này cho ghi chú triển khai, ghi chú cấu trúc và các file hỗ trợ của khu vực này.
