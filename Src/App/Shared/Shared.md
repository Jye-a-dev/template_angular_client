# Shared

## English
Purpose: This folder defines the Shared area inside Shared.

Current contents:
- ``Components/``: subfolder for components related code and docs.
- ``Directives/``: subfolder for directives related code and docs.
- ``Pipes/``: subfolder for pipes related code and docs.

Use this folder for:
- Store files related to this area here.
- Keep this concern organized and separate from unrelated code.
- Use this folder for implementation notes, structure notes, and supporting files for this area.

## Tiếng Việt
Mục đích: Thư mục này định nghĩa khu vực Shared bên trong Shared.

Nội dung hiện tại:
- ``Components/``: thư mục con cho mã nguồn và tài liệu liên quan đến components.
- ``Directives/``: thư mục con cho mã nguồn và tài liệu liên quan đến directives.
- ``Pipes/``: thư mục con cho mã nguồn và tài liệu liên quan đến pipes.

Nên dùng thư mục này cho:
- Chứa các file liên quan đến khu vực này tại đây.
- Giữ concern này được tổ chức rõ ràng và tách khỏi phần code không liên quan.
- Dùng thư mục này cho ghi chú triển khai, ghi chú cấu trúc và các file hỗ trợ của khu vực này.
